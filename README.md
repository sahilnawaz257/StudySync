# Library Attendance System

A comprehensive system for tracking library attendance using QR codes.

## Project Structure

- **client/**: React-based frontend application.
- **server/**: Express and Prisma-based backend application.

## Features

- Student Registration and Login
- QR Code Generation and Scanning
- Attendance Tracking and Reporting
- Admin Dashboard

## Getting Started

### Prerequisites

- Node.js (v18+)
- PostgreSQL (with Prisma)

### Backend Setup

1. `cd server`
2. `npm install`
3. `npx prisma generate`
4. `npm run dev`

### Frontend Setup

1. `cd client`
2. `npm install`
3. `npm run dev`
